File format¶

  • Lines 1-3: [ax_i] [ay_i] [az_i]

  • Line 4: [Norbit]

  • Lines 5-: [vx_i] [vy_i] [vz_i]

Parameters¶

  • [ax_i], [ay_i], [az_i]

    Type : Float

    Description : These parameters for i from 1 to 3 specify the primitive vectors \(\vec{a}_1, \vec{a}_2, \vec{a}_3\).

  • [Norbit]

    Type : Integer

    Description : This parameter specifies the number of orbitals \(N_\text{orbit}\) in a unitcell.

  • [vx_i], [vy_i], [vz_i]

    Type : Float

    Description : These parameters specify the Wannier center \(\vec{v}_i\) of each orbital in the fractional coordinates.

Usage rules¶

  • The indices of the orbitals are implicitly assigned from 1 to \(N_\text{orbit}\) in the order of the Wannier centers.
