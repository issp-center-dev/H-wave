ファイル形式¶

  • 1-3行: [ax_i] [ay_i] [az_i]

  • 4行: [Norbit]

  • 5行以降: [vx_i] [vy_i] [vz_i]

パラメータ¶

  • [ax_i], [ay_i], [az_i]

    形式 : float型

    説明 : 基本格子ベクトル \(\vec{a}_1, \vec{a}_2, \vec{a}_3\) を指定します。

  • [Norbit]

    形式 : int型

    説明 : ユニットセル内の軌道の数 \(N_\text{orbit}\) を指定します。

  • [vx_i], [vy_i], [vz_i]

    形式 : float型

    説明 : 各軌道のWannier center \(\vec{v}_i\) を fractional coordinate 表記で指定します。

使用ルール¶

本ファイルを使用するにあたってのルールは以下の通りです。

  • 軌道のインデックスは暗黙的に Wannier center の並び順に 1〜\(N_\text{orbit}\)で与えられます。
